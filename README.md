## :bulb: Project : 딥러닝을 활용한 마스크 착용여부 탐지 프로그램 구현
최근 코로나-19의 재 확산으로 마스크 착용이 다시 필수가 되어 가고 있다. 따라서 본 프로젝트에서는 마스크 착용 여부 감지를 딥러닝을 이용한 프로그램으로 자동화 함으로서 인력 낭비 문제를 해결한다. 
## :hammer: Project Tool
* Python
* YOLOv5
* OpenCV
